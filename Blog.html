Frequently Asked Questions

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web pages. It represents the structure of a page, allowing programs to modify its content, style, and layout. By organizing the document into nodes and objects, the DOM enables programming languages to interact with and manipulate the page dynamically.

How do you select an element from the DOM?

I can select an element from the DOM using methods like document.getElementById(), document.querySelector(), document.querySelectorAll() , document.getElementsByClassName() or document.getElementsByTagName().


document.getElementById('id') : Selects an element by its id.


document.querySelector('selector') : Selects the first element that matches a CSS selector.


document.querySelectorAll('selector') : Selects all elements that match a CSS selector.


document.getElementsByClassName('className') : Selects all elements with a given class name.


document.getElementsByTagName('tagName') : Selects all elements with a given tag name.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events for multiple child elements, instead of adding individual listeners to each child.


It is useful because it reduces memory usage and improves performance by minimizing the number of event listeners attached to multiple elements. It also handles events for dynamically added child elements, as the parent listener will still capture events from newly created children.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an element's attributes and styles using the properties and methods provided by the DOM.

For example, I can change an element's text using the innerText or textContent properties, and modify its attributes with the setAttribute() and getAttribute() methods. Additionally, I can adjust its styles using the style property or manipulate classes with classList.add() and classList.remove().
